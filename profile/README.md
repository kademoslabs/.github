# Kademos Labs

We build lean security and compliance tools that help teams ship.

## Products
- **Kekkai CLI** — one-command AppSec orchestration and clean reports.  
  → https://kademos.org/kekkai  
  → Repo: https://github.com/kademoslabs/kekkai-cli

- **Regulon** — PSTI compliance tooling for UK product sellers and importers.  
  → https://kademos.org/regulon

- **Responsa** — security questionnaire acceleration for sales teams.  
  → https://kademos.org/responsa
